# Alibaba Scrapper
Scrapes alibaba for searched terms.

## Prerequisites
* [python 2.7](https://www.python.org/download/releases/2.7/)
* [pip](https://pypi.python.org/pypi/pip)

## How to run

1. Install required packages
```
pip install -r requirements.txt
```

2. Run
```
python srp.py
```
